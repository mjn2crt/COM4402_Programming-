## Project Overview
This project is a quiz application developed in Python  for the Holton College 
Digital Quiz System. The application presents multiple-choice questions to the 
user, accepts input, checks answers, keeps score, and displays the final result 
at the end of the quiz

## Features
- Stores quiz questions using appropriate Python data structures
- Displays questions and multiple-choice options one at a time
- Accepts user input for answers
- Validates answers and updates the score
- Displays final score and a closing message
- Clear, readable, and well-commented code
